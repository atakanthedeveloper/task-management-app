# Task Management App

A responsive task management application built with **HTML**, **CSS**, and **Vanilla JavaScript**.  
This project allows users to create, edit, complete, delete, filter, and sort tasks while keeping data persistent with **localStorage**.

## Overview

This project was developed as a frontend practice application to improve JavaScript fundamentals such as:

- DOM manipulation
- Event handling
- Conditional rendering
- State-based UI updates
- localStorage usage
- Dynamic filtering and sorting logic

## Features

- Add new tasks with category and priority
- Edit task name, category, and priority
- Save or cancel task changes
- Mark tasks as completed
- Delete individual tasks
- Clear all tasks
- Clear completed tasks
- Filter tasks by:
  - All
  - Active
  - Completed
- Sort tasks by:
  - Newest
  - Oldest
  - Priority
  - Category
- Store tasks in localStorage
- Show active task count dynamically
- Highlight priority levels with color-coded options
- Responsive layout for smaller screens

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage
